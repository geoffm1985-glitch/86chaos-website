What you still need to send

You do not need seven perfect screenshots anymore.

The Product Tour now uses polished demo panels. Real screenshots can be added later only if they are cleaner than the demo panels and show populated demo data without errors or private information.

Only send screenshots if they improve trust.

Good: clean demo data, no browser chrome, no Firebase errors, no real staff info, no empty states.

Skip: settings errors, account security warnings, debug panels, and anything that looks half-built.
